# WhatsApp Green Chat

### Install
Use `npm install`, `yarn install` or `bun install` to install dependencies

### Local run
Run `npm run dev` or `yarn dev` or `bun dev` to start the app locally.

### Build
To build use `npm run build` or `yarn build` or `bun build`
